LBM are very efficients and can easily exploit the heterogeneous hardware available today.


LBM can also deal with complex geometries and are used in various areas: porous media, multiphase flows, ...

Pros

  • Complex geometry
  • Performance
  • Adaptive mesh refinement

Cons

  • Hard to define new schemes
  • No mathematical tools for LBM analysis

But...

LBM are different from conventional CFD methods and if you are a beginner, it can be


  • Hard to understand what physical equations are solved
  • Hard to fix the parameters (relaxation, equilibrium, ...)
  • Hard to implement real boundary conditions (Dirichlet, Neumann, ...)

pylbm tries to simplify this whole process.

Other features of pylbm

  • Highly flexible code generation
  • Parallelization using MPI
  • GPU support using loopy
  • STL support for complex geometries
  • Stability analysis of your scheme
  • Adaptive mesh refinement using samurai (work in progress)
